Pozdravljeni!


Ta stran je namenjena učenju in preverjanju znanja stavkov SQL.


Za vas smo pripravili povzetek vsega, ki lahko služi kot učenje oziroma ponovitev vašega znanja.


Select


Izjava SELECT se uporablja za pridobivanje informacij iz podatkove baze iz ene ali več tabel in vrne podatke v obliki tabele.


Stavek SELECT se uporablja, da pridobimo eno ali več vrstic podatkov, ki se ujemajo s podanimi kriteriji iz podatkovne baze.


Primer SELECT stavka, ki nam izpiše ime in priimek vseh študentov:


SELECT firstname, lastname


FROM student;


Agregatne funkcije


Poznamo več agregatnih funkcij, ampak mi bomo spoznali samo naslednje, ki se uporabljajo največ:


Funkcija COUNT


Funkcija COUNT nam vrne število elementov, ki ustrezajo pogojem.


Primer uporabe COUNT pri številu vseh tečajev:


SELECT COUNT (*)


FROM course;


Funkcija AVG


Funkcija AVG nam vrne povprečno vrednost, ki ustreza določenim pogojem.


Primer uporabe AVG pri povprečni ceni tečajev:


SELECT AVG (price)


FROM course;


Funkcija MIN


Funkcija MIN nam vrne najmajnšo vrednost, ki ustreza določenim pogojem.


Primer uporabe MIN pri najmanjši ceni tečaja:


SELECT MIN (price)


FROM course;


Funkcija MAX


Funkcija MAX nam vrne največjo vrednost, ki ustreza določenim pogojem.


Primer uporabe MAX pri največji ceni tečaja:


SELECT MAX (price)


FROM course;


Join


Če želimo dostopati do več kot ene tabele naenkrat moramo znati uporabiti JOIN.


Poznamo več različnih vrst združevanja ampak za nas bo dovolj naj preprostejši JOIN.


Če želimo dostopati do več tabel hkrati moramo vedeti kako so tabele med seboj povezane.


To lahko razberemo iz E-R modela, ki je prikazan zraven vsakega vprašanja.


Primer uporabe JOIN pri poizvedbi za prikaz države in pošte:


SELECT *


FROM student


JOIN address ON student.FK_address = address.id;


Group by in Order by


Z uporabo GROUP BY besede razdelimo vrstice, ki jih dobimo iz stavka SELECT v skupine.


Največkrat se uporabljajo z agregatnimi funkcijami, za izračun števila elementov v skupini in podobno.


Z uporabo ORDER BY besede pa razvrstimo rezultate po abecednem vrstnem redu. Ali po obratnem vrstnem redu z besedo DECS


Primer uporabe ORDER BY in GROUP BY pri izpisu države in števila pošt po vrstnem redu.


SELECT state.name, count(mail.FK_state)


FROM state


JOIN mail ON mail.FK_state = state.id


GROUP BY state.name


ORDER BY state.name;


Having


Beseda HAVING se največkrat uporablja kot iskalni pogoj za skupino.


Pogosto se uporablja z GROUP BY za filtriranje na podlagi določenega pogoja.


Primer uporabe HAVING in GROUP BY za izpis vseh študentov z imenom 'Luka'.


SELECT *


FROM student


GROUP BY id


HAVING firstname = 'Luka';
